# Newelle-WebUI

This is a vibe-coded WebUI for Newelle. This is used to show the capabilities and to test the `gui-api` interface.
**Note**: This project should be considered as an unofficial extension of Newelle

- [x] Basic chat management

- [x] Speech To Text
- [x] Text To speech
- [x] TTS Settings
- [x] Profile Switching
- [ ] Image/Vision support
- [ ] Kowledge settings
- [ ] Interface settings
